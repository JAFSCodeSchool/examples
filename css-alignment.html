Now lets see Vertical Alignment. with the above example.

Vertical alignment works only if you specify a height to the outer element. Now to all the inner elements I am using margin to align them vertically.
- To the first div i have given margin bottom as auto. Which means, what all spaces is left will be set to bottom of this div.
- To the second div i have given margin top as auto. Which means, what all spaces is left will be set to top of this div.
- To the third div i have given margin top and bottom as auto. Which means, what all spaces is left will be divided equally between top and bottom.
Note: If you dont specify margin to the inner div's, by default it will have full height. After setting up the margin that full height will go and element height will work based on the size of the inner content.
